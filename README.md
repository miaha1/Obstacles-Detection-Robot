# Obstacles-Detection-Robot
Robot Path Planning While Detecting Obstacles
## Overview
This project focuses on implementing and testing Rapidly-Exploring Random Trees (RRT) for robot path planning. The goal is to develop an efficient algorithm for navigating through complex environments.

---

## Key Files

- **`cmap.py`**: Provides utilities for handling configuration maps.
- **`gui.py`**: Implements a graphical user interface for visualizing RRT.
- **`rrt.py`**: Core implementation of the RRT algorithm.
- **`maps/`**: Directory containing map configurations for testing:
  - `emptygrid.json`

---

## How to Run

1. **Set up the environment:**
   - Environment Requirement:
    python 3.9 or below. 

3. **Run the scripts:**
   - To test the RRT implementation:
     ```bash
     python rrt.py
     ```

---

## Notes
- Ensure all dependencies are installed before running the scripts.
- Modify the map configurations in the `maps/` directory as needed for custom testing.
